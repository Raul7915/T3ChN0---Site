Care este diferent​a si ce inseamna: Chip, Chipset, Processor, CPU

      Am ajuns la conculiza ca de multe ori ne aruncam in tutoriale complicate sau discutii tehnice, insa foarte multi nu stiu ce inseamna sau folosesc gresit termeni precum: chip, CPU, processor si chipset.
      Foarte multi si-au format ideea gresita ca “chip” este totuna cu “CPU” si “Processor“, iar termenul “chipset” se refera la setarile chip-ului. Nimic mai gresit.Care este diferenta dintre “Chip” si “CPU”?In primul rand, “CPU” este totuna cu “Processor”. Central Pocessing Unit (CPU) este o componenta hardware de baza, fara ca de care un calculator (sistem informatic) nu poate functiona. Impreuna cu memoria RAM, unitatea de sotcare date (hard disk), placa de baza (motherboard), circuite si componente auxiliare, alcatuiesc un sistem informatic functional.
      Atunci cand vrem sa cumparam un PC, cautam de multe ori ca acesta sa fie prevazut cu un CPU cat mai puternic. Aceasta putere a CPU se traduce prin frecventa / viteza de executie a operatiunilor aritmetice si logice pe care le primeste de la o aplicatie sau de la sistemul de operare. Unitatea de masura ce stabileste performanta unui procesor, este MHz sau GHz. Unitate de masura universala pentru frecventa.

Un “chip” sau “microchip” este un dispozitiv electronic compus din unul sau mai multe circuite. Un chip poate fi compus de exemplu, din unul sau mai multe CPU, dar poate sa contina si alte circuite pentru diverse operatiuni. Nu este obligatoriu ca un chip sa contina CPU. Tranzistori, diode, condensatoare, blocuri logice, emitatoare si receptoare de semnal (RF), plus multe altele care pot fi incluse intr-un chip.

      Asadar, “chip” si “CPU” sunt doi termeni ce definesc componente total diferite intr-un sistem informatic.Termenul “chipset” are o stransa legatura atat cu “CPU” cat si cu “chip”, insa nu poate fi atribuit uneia dintre aceste componente.
Chipset-ul este componenta a placii de baza (motherboard) ce conecteaza si coordoneaza fluxul de date si instructiuni dintre CPU si celelalte componente hardware esentiale ce compun sistemul informatic. Tot de chipset tine si coordonearea dintre CPU si dispozitivele externe. Un chipset impecabil este de dorit pe orice placa de baza pentru a atinge performante de top.
